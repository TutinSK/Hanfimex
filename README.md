# Product Showcase Website

Trang web tĩnh giới thiệu sản phẩm được xây dựng bằng React + Vite.

## 🚀 Tính năng

- **Header**: Menu điều hướng với thông tin liên hệ
- **Hero Section**: Phần giới thiệu chính với slogan
- **6F Concept**: Fresh Farm - Fair Factory - Fine Food
- **Why Choose Us**: Lý do chọn sản phẩm
- **Products Showcase**: Hiển thị các sản phẩm chính
- **Footer**: Thông tin liên hệ và mạng xã hội

## 📦 Cài đặt

```bash
npm install
```

## 🛠️ Chạy Development Server

```bash
npm run dev
```

Mở trình duyệt tại: http://localhost:5173/

## 🏗️ Build cho Production

```bash
npm run build
```

## 🎨 Tùy chỉnh

Bạn có thể chỉnh sửa:
- Tên thương hiệu, slogan trong các component
- Màu sắc chủ đạo trong file CSS
- Danh sách sản phẩm trong Products.jsx
- Thông tin liên hệ trong Footer.jsx
